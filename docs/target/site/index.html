    lagom-pb

    lagom-pb = lagom + Protocol Buffer

    Scala shared code for lagom development using protocol buffer.

    This library helps write lagom based microservices easily by making use of protocol buffer messages to define the es/cqrs core components like api requests/responses, grpc services, events, commands and state.
